1. Na co pozwala styl lokalny, który element będzie formatowany z użyciem stylu lokalnego?

    2. Styl lokalny, zwany też inline, pozwala na formatowanie konkretnego elementu HTML poprzez dodanie atrybutu style bezpośrednio do tego elementu. Chociaż jest to szybkie rozwiązanie, często odradzane jest jego używanie ze względu na problemy z utrzymaniem i skalowalnością kodu.

  3. Co może być selektorem?

    4. Selektorem może być praktycznie dowolny znacznik, np. p (akapit), h1 (nagłówek), td (komórka tabeli) i inne. W języku CSS selektor to element, który wskazuje, jakie elementy HTML mają być stylizowane.

  4. Co wipisujemy jako "cechę"? Jakie jest znaczenie średników?

    5. W CSS jako "cechę" wpisujemy właściwość (ang. property), która określa, jaki aspekt stylowania elementu chcemy zmienić. Średnik (;) w CSS służy do oddzielania kolejnych deklaracji w bloku stylów. Każda para właściwość: wartość powinna kończyć się średnikiem, aby przeglądarka poprawnie interpretowała kod.

  5. Co wipisujemy jako "wartość" (w deklaracji stylu)?

    6. W deklaracji stylu "wartość" to konkretne ustawienie dla danej właściwości (cechy). Może to być: Kolor, Liczba z jednostką, Słowo kluczowe, Adres USL, Wartości wielokrotne, Funkcje CSS.
